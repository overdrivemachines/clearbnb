# Clearbnb

This is a clone of Airbnb

![Preview](preview.png)

Live - https://clearbnb.dipenchauhan.com

# Features

- Authentication
- CRUD Listings
- Way to Book
  - Payment
- Messaging
  - Canned responses
  - Scheduled messages
  - 2 way messaging between host and guest
- ID Verification
- Calendar System
